# OpenRoad.am — Концепция платформы

> Интерактивная карта дорожных проблем Армении с AI-распознаванием и открытым API

---

## Миссия

Дать каждому жителю Армении простой инструмент для сообщения о дорожных проблемах — и сделать так, чтобы эти данные реально доходили до тех, кто может их исправить.

---

## Проблема

Дороги Армении остаются одними из наиболее опасных в регионе: более 300 смертей в год, выбоины, отсутствующая разметка, сломанные знаки. При этом:

- Граждане не имеют удобного канала для жалоб
- Госорганы не получают структурированных данных от населения
- Не существует публичной карты актуальных проблем
- Нет инструмента для отслеживания — была ли проблема устранена

---

## Решение

Веб- и мобильная платформа, где любой пользователь может:

1. **Сфотографировать** проблему на дороге
2. **Получить AI-классификацию** типа повреждения (выбоина, отсутствие разметки, сломанный знак, опасный участок и т.д.)
3. **Отметить точку** на интерактивной карте
4. **Отследить статус** — принято ли обращение, взято ли в работу

Данные агрегируются, верифицируются и передаются в профильные госорганы через открытый API.

---

## Аудитория

### Водители и жители
Хотят быстро сообщить о проблеме. Минимум действий: сфоткал → отправил → готово. Видят, что их репорт не исчез в никуда.

### Журналисты и активисты
Используют данные для расследований и публикаций. Им важен открытый API, экспорт данных, аналитика по районам и динамика во времени.

### Госорганы
Получают агрегированные, верифицированные данные без спама. Дашборд с приоритизацией проблем по частоте репортов и опасности.

## Конкурентная среда

| Решение | Что делает | Чего не хватает |
|---|---|---|
| Arm Roads Info | Репорты об авариях и дорожных условиях | Нет AI, нет открытого API |
| UNDP SDG Platform | Аналитика чёрных точек для госорганов | Закрытая, не для граждан |
| Закон о репортах нарушений ПДД | Гражданские жалобы на нарушителей | Только нарушения, не состояние дорог |

**Вывод:** ниша публичной краудсорсинговой карты с AI и открытым API свободна.

---

## Технология

- **Интерактивная карта** — публичная, обновляется в реальном времени
- **AI-распознавание** — классификация типа проблемы по фото
- **MCP-интеграция** — работа с AI-инструментами через открытый протокол
- **Открытый API** — любые разработчики, СМИ и госорганы могут подключиться
- **Open source код** — MIT лицензия, доверие сообщества

---

## Бизнес-модель

**Текущий этап**
- Рекламные баннеры (минимально, нативно) — покрывают операционные расходы

**Среднесрочно**
- Гранты: UNDP Armenia, EU4Armenia, USAID, Open Society Foundations
- B2G: платный доступ к верифицированным данным для MTAI и муниципалитетов

**Долгосрочно**
- B2B: данные для страховых компаний (оценка рисков по маршрутам)
- API-подписки для коммерческих пользователей

---

## Юридическая структура

Коммерческое ИП (Армения) + открытый код. Гибридная модель: социальное позиционирование без ограничений НКО. Позволяет подавать на гранты, заключать B2G контракты и зарабатывать коммерчески.

---

## Стратегия выхода на госорганы

### Фаза 1 — Данные (3–6 месяцев после запуска)
Набрать критическую массу репортов. Целевой минимум — 500–1000 репортов по Еревану.

### Фаза 2 — Медиа-давление
Публикации в армянских СМИ (1in.am, Hetq, CivilNet) на основе данных платформы. Создать публичный контекст перед переговорами.

### Фаза 3 — Переговоры
Целевые ведомства:
- **MTAI** (Министерство территориального управления и инфраструктуры) — межгосударственные трассы
- **Мэрия Еревана** — городские улицы
- **Бюро автостраховщиков** — коммерческий партнёр

Формат предложения: *"Мы уже собрали X тысяч репортов. Отдаём верифицированные данные через API — вам не нужно ничего разрабатывать."*

---

## Тон и ценности

Платформа позиционируется как **конструктивный инструмент**, а не инструмент критики. Не "посмотрите какие ужасные дороги" — а "вот данные, чтобы чинить быстрее и точнее".

Это критично для доверия госорганов и для грантового позиционирования.
